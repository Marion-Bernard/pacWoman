## Description
This project is a personal learning project made for fun.  
The goal is to recreate the core mechanics of the classic Pac-Man game, such as movement, scoring and win/lose conditions.

It is not intended to be a full or commercial version, but a way to practice game logic and front-end development.

## Technologies
- HTML5
- CSS3
- JavaScript
- Canvas
- Tailwind CSS (CDN)

## Features
- Player movement
- Walls
- Collectible points
- Score display
- Win and Game Over screens
- Restart game functionality
